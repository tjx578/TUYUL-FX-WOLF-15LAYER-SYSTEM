# MT5 Web and Terminal Usage

**Type:** Platform knowledge

## Purpose

Summarize how MT5 surfaces are used operationally without confusing them with system authority.

## Key boundary

MT5 is an execution venue and monitoring touchpoint. It is not the strategic brain of the system.

## Safe usage model

- monitor fills and broker responses
- confirm account state when needed
- validate bridge health and symbol availability
- avoid manual actions that bypass governance and pending-order rules
